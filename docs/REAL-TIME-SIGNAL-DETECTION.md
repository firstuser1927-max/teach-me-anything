# Real-Time Signal Detection

```
┌─────────────────────────────────────────────────────────────────────┐
│                REAL-TIME SIGNAL DETECTION                           │
├─────────────────────────────────────────────────────────────────────┤
│                                                                     │
│  During a lesson, watch for these signals in the user's responses:  │
│                                                                     │
│  CONFUSION SIGNALS                                                  │
│  ──────────────────                                                  │
│  ┌─────────────────────────────────────────────────────────────┐   │
│  │ "I don't understand"                                       │   │
│  │ "Can you explain that differently?"                        │   │
│  │ "What does X mean?" (about a term already defined)         │   │
│  │ "That doesn't make sense"                                  │   │
│  │ Silent for a long time, then a wrong answer                │   │
│  │ Asks question that shows they misread the explanation      │   │
│  │                                                             │   │
│  │ → ACTION: Reframe using different representation           │   │
│  │ → RECORD: Add confusion instance to LEARNER-PROFILE        │   │
│  └─────────────────────────────────────────────────────────────┘   │
│                                                                     │
│  BOREDOM SIGNALS                                                    │
│  ────────────────                                                    │
│  ┌─────────────────────────────────────────────────────────────┐   │
│  │ "I already know this"                                      │   │
│  │ "Can we skip to the exercise?"                             │   │
│  │ "Yeah yeah, got it" (before seeing practice)               │   │
│  │ Answers correctly without reading the knowledge section    │   │
│  │ Asks to move faster                                        │   │
│  │                                                             │   │
│  │ → ACTION: Skip remaining scaffolding, go to challenge      │   │
│  │ → RECORD: Note pace may be faster than calibrated          │   │
│  └─────────────────────────────────────────────────────────────┘   │
│                                                                     │
│  FRUSTRATION SIGNALS                                                 │
│  ───────────────────                                                 │
│  ┌─────────────────────────────────────────────────────────────┐   │
│  │ "This is too hard"                                         │   │
│  │ "I can't do this"                                          │   │
│  │ "Why isn't this working?" (repeatedly)                     │   │
│  │ Short, clipped responses                                    │   │
│  │ Gives up on a question without trying                      │   │
│  │ "Maybe I should just stop for today"                       │   │
│  │                                                             │   │
│  │ → ACTION: Pause. Validate feeling. Offer simpler path.     │   │
│  │ → RECORD: Update frustration threshold in profile            │   │
│  │ → CRITICAL: Never push through frustration. Stop or ease.   │   │
│  └─────────────────────────────────────────────────────────────┘   │
│                                                                     │
│  FLOW SIGNALS (positive)                                             │
│  ─────────────────────                                               │
│  ┌─────────────────────────────────────────────────────────────┐   │
│  │ "Oh, I see!"                                               │   │
│  │ "That makes sense now"                                     │   │
│  │ Answers correctly and adds own example                     │   │
│  │ Asks a "what if?" question (extending the concept)         │   │
│  │ Teaches back: "So basically it's like..."                   │   │
│  │                                                             │   │
│  │ → ACTION: This is mastery signal. Can accelerate.           │   │
│  │ → RECORD: Note as evidence of deep understanding           │   │
│  └─────────────────────────────────────────────────────────────┘   │
│                                                                     │
│  FATIGUE SIGNALS                                                    │
│  ────────────────                                                    │
│  ┌─────────────────────────────────────────────────────────────┐   │
│  │ Answers start getting sloppy after correct ones             │   │
│  │ "Let me come back to this later"                            │   │
│  │ Takes much longer to respond                                │   │
│  │ Stops asking questions (even when they should)              │   │
│  │                                                             │   │
│  │ → ACTION: Wrap up current lesson, don't start another      │   │
│  │ → RECORD: Update session capacity in profile               │   │
│  └─────────────────────────────────────────────────────────────┘   │
│                                                                     │
└─────────────────────────────────────────────────────────────────────┘
```
