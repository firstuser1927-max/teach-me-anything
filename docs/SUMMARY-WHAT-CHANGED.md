# Summary: What Changed

| Component | Before | After |
|-----------|--------|-------|
| Learner model | None (implicit) | Explicit LEARNER-PROFILE.md inferred from behavior |
| Difficulty | One-size ZPD | Calibrated by last 3 lesson outcomes (-3 to +3 score) |
| Scaffolding | Implicit in lessons | 3 explicit levels, selected per lesson |
| Representation | Author's choice | Adapted to learner preference, switched on struggle |
| Error handling | "Explain if wrong" | Pattern-classified (careless/gap/transfer/overgen) with strategies |
| Real-time | None | Signal detection (confusion/boredom/frustration/flow/fatigue) |
| Session pacing | Undefined | Tracked capacity, fatigue-aware stopping |
| Records | "What was learned" | + outcome classification, signals, adaptations |

**Core principle:** The learner profile is **inferred, never asked.** People are bad at knowing how they learn. Their behavior tells the truth.
